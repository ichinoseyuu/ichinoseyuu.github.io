这次有思路，但是写了好久时间，以下是具体思路。

  • 函数首先判断如果列表只有一个元素，直接返回 1。
  • 然后创建一个栈对象 stack，用于存储已经遍历过的元素。初始化结果变量 result 为 0，临时变量 temp 为 1，表示当前正在计算的不重复子数组的长度。
  • 接下来通过循环遍历数组的前 n-1 个元素（n 为数组长度）。在每次循环中，将当前元素压入栈中，然后判断栈中是否包含下一个元素 nums[i+1]。如果不包含，则说明下一个元素和当前元素不重复，此时 temp 加 1，并更新 result 为 temp 和 result 的较大值。否则，说明下一个元素和当前元素重复，此时需要更新 result 为 temp 和 result 的较大值，并将 temp 重置为 1，同时清空栈。
  • 最后，返回结果 result 即可。
