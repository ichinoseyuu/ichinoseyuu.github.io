一、环境搭建

查看系统版本支持

选择 Tensorflow 版本

下载 cuda11.0 及 cuDNN8.0，安装 cuda11.0，并将 cuDNN 解压放入 cuda 安装文件夹

检查是否配置成功

用 Anaconda 配置虚拟环境 Tensorflow24

检查是否安装成功

安装 opencv-python==4.5.5.62

Anaconda 库中并没有收录该版本包，采用本地包安装

安装 numpy==1.19.5

安装 Pillow==8.4.0

安装 tensorboard==2.7.0

安装 tqdm

从 github 将 Tensorflow-yoloV6 项目克隆到本地
